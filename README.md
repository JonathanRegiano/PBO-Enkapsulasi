Menggunakan tiga class yaitu Student, Grade, dan PBOEnkapsulasi. 
Class Student digunakan untuk menyimpan dan mengelola informasi dasar mahasiswa. Class ini memiliki atribut studentId, name, dan age yang dibuat sebagai private.
Class Grade digunakan untuk mengelola data nilai mahasiswa. Class ini memiliki atribut score1, score2, dan score3 yang juga dibuat sebagai private.
Class PBOEnkapsulasi merupakan class utama yang digunakan untuk menjalankan program. Pada class ini dibuat objek dari class Student dan Grade untuk mengisi data mahasiswa dan nilai yang dimiliki.
